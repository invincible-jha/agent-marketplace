# Examples

| # | Example | Description |
|---|---------|-------------|
| 01 | [Quickstart](01_quickstart.py) | Minimal working example with the Marketplace convenience class |
| 02 | [Capability Registration](02_capability_registration.py) | Register capabilities with metadata and quality metrics |
| 03 | [Discovery and Search](03_discovery_search.py) | Find capabilities by keyword, category, and constraints |
| 04 | [Trust and Reviews](04_trust_reviews.py) | Submit reviews and compute provider trust scores |
| 05 | [Matching and Negotiation](05_matching_negotiation.py) | Match capabilities to requests and negotiate prices |
| 06 | [Tool Catalog](06_tool_catalog.py) | Register and discover tools with the ToolCatalog |
| 07 | [LangChain Marketplace](07_langchain_marketplace.py) | Discover capabilities and load as LangChain tools |

## Running the examples

```bash
pip install agent-marketplace
python examples/01_quickstart.py
```

For framework integrations:

```bash
pip install langchain   # for example 07
```
